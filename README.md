# LoginForm-hansdon

A simple **login form** project built using **HTML, CSS, and JavaScript**, featuring **client-side validation**.

## Features

- Responsive login form
- Client-side validation:
  - Email and password cannot be empty
  - Password must be at least 8 characters long
- Prevents form submission if validation fails
- Displays form data in console as JSON on successful submission

## Technologies Used

- **HTML** for the form structure
- **CSS** for responsive layout and design
- **JavaScript** for form validation and interactivity
